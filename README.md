# 🚀 Portfolio Website (Next.js)

> 💡 **SEO Optimization Notice:** For full documentation on how this project ranks #1 on search engines for its owner, please reference the [SEO-GUIDE.md](./SEO-GUIDE.md) located in the project's root folder.

A modern, highly interactive personal portfolio styled with a sleek, developer-focused **terminal theme**. Built using **Next.js** and **React**, it cleanly showcases low-level engineering projects, skills, and professional experience. The site includes a fully functional terminal contact form subsystem powered by the **Resend API**, enabling seamless email communication.

---

## ✨ Overview

This project is a production-ready developer portfolio designed to be fast, responsive, and easily maintainable. It emphasizes a unique command-line aesthetic (complete with directory structures and simulated cursors), modular components, and scalable architecture using modern frontend tools.

---

## 🧱 Tech Stack

### Frontend
- **Next.js (App Router)**
- **React**
- **TypeScript**
- **Tailwind CSS** (Configured with terminal colors: dark backgrounds, bright accents, and a flashing monospaced block cursor)

### Backend / API Layer
- **Next.js API Routes** (Serverless execution environment)
- **Resend API** for email delivery (Contact form handler system)

### Deployment
- **Vercel** (Optimized edge caching for next-gen performance indices)

---

## 📦 Core Features

### 🧑‍💻 Portfolio Sections (Terminal Themed)
- **Interactive Directory Simulation:** Sections styled as shell environments (`/projects:~$ █`, `/experience:~$ █`, `/skills:~$ █`).
- **Hero / Landing Shell:** Monospaced introduction layout highlighting computer engineering specialties.
- **Projects Showcase:** Dynamic grid layouts mirroring real-time interactive terminal card components.
- **`README.md` About Box:** A custom component styled like a code repository file editor that allows collapsible deep-dives into professional biographies.

### 📬 Contact System
- Built using **Resend API**
- Server-side email handling via secure API routes
- Form validation before submission
- Anti-spam-friendly request structure

### ⚡ Performance & UX
- Server-side rendering (SSR) / static generation (SSG)
- Fully responsive design (Mobile-first terminal window views)
- Built-in Next.js font rendering engines loaded with high-legibility developer typography
- Smooth CSS keyframe console animation transitions
